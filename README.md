# CEVAR: Centerline Embedding Extraction for Endovascular Aneurysm Repair
This repository provides the official implementation of the CEVAR: Centerline Embedding Extraction for Endovascular Aneurysm Repair paper by [Roman Naeem](https://scholar.google.com/citations?user=6fVdbR0AAAAJ), [Timo Niiniskorpi](https://scholar.google.com/citations?user=PHE0sFMAAAAJ), [Naman Desai](https://scholar.google.com/citations?user=XrInmKAAAAAJ), Charlotte Sandström, [Anders Jeppsson](https://scholar.google.com/citations?user=CIvP2lEAAAAJ), [Ida Häggström](https://scholar.google.com/citations?user=Bh4IQ30AAAAJ), [Fredrik Kahl](https://scholar.google.com/citations?user=P_w6UgMAAAAJ&hl=en), Håkan Roos and [Jennifer Alvén](https://scholar.google.com/citations?user=QmYcKQQAAAAJ).
